## How to run   
### Prerequisites
- NVIDIA GPU + CUDA CuDNN (CPU untested, feedback appreciated)
- Pytorch

```bash
# clone project   
git clone https://github.com/yjzhang96/UTI-VFI 
cd UTI-VFI
 ``` 
### Dataset
download GoPro datasets, and generate blurry videos for different exposure settings.
   

### Test
 ```bash
sh run_test.sh
```
